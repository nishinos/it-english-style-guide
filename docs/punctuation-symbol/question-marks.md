# 疑問符（?）

疑問符（?）についての説明です。

## 疑問符を使う場面

疑問符は、疑問文の末尾で使います。

**例**

> Already added Firebase to your C++ project? Make sure that you're using the latest version of the Firebase C++ SDK.
> 
> （引用元：[Firebase](https://firebase.google.com/docs/cpp/setup?platform=ios)、2023-02-15閲覧）
